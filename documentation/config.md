# Configuration Setup 

Refer to this document for general examples for configuration file setup. If finding that there are issues with the .gitignore and the config.ini refer to the git_repo markdown.

Default Setup can be found in the `config.ini.example` it is critical that a copy of this file be made into the config.ini on any new devices (This may be automatically done in future versions of these systems).

Default:

[general]
name: piX
test: n

[imaging]
w = 2304
h = 1296
lens_position: 2.0
startup: 5,0,0
shutdown: 16,0,0
